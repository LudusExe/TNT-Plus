**TNT Plus** is a Voxelibre/Mineclonia mod heavily inspired by the classic Minecraft mod *Too Much TNT*; the uranium system was initially a separate mod. However, it has been incorporated into *TNT Plus*, as it is a natural combination.

## Features

### More TNT

* **Freeze TNT**
* **Lava TNT**
* **Water TNT**
* **Flat TNT**
* **Flat Mountain TNT**
* **Vertical TNT**
* **House TNT**
* **Tent TNT**
* **Rain TNT**
* **Airstrike TNT**
* **Mob TNT**
* **Atomic TNT**
* **Black Hole TNT**
* *TNT X2**
* **TNT X5**
* **TNT X20**
* **TNT X100**
* **TNT X500**

## Uranium Stuff

Materials:

* Uranium Ore
* Uranium Dust
* Uranium Ingot
* Uranium Block (may explode when mined)

Full armor and tool set:

* Sword
* Axe
* Pickaxe
* Shovel
* Hoe

Provides protection like the netherite set, but with occasional explosions.
